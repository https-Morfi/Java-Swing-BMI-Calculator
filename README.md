# Java-Swing-BMI-Calculator
این پروژه یک اپلیکیشن گرافیکی ساده و کاربردی برای محاسبه شاخص توده بدنی (BMI) است که با استفاده از Java Swing پیاده‌سازی شده است. هدف این برنامه ارائه یک تجربه کاربری روان، قابل فهم و در عین حال حرفه‌ای برای محاسبه BMI، نمایش وضعیت بدنی و ارائه پیشنهادات سلامتی متناسب با شرایط هر کاربر است.

در این برنامه:

BMI با دقت دو رقم اعشار محاسبه می‌شود

نتیجه در یکی از دسته‌های زیر قرار می‌گیرد:

کمتر از 18.5 → کم‌وزن (Underweight)

بین 18.5 تا 24.9 → نرمال (Normal)

بین 25 تا 29.9 → اضافه‌وزن (Overweight)

30 و بیشتر → چاق (Obese)

وضعیت بدنی با رنگ‌بندی مناسب در رابط کاربری نمایش داده می‌شود تا درک نتیجه برای کاربر ساده‌تر شود.


ساختار کلی برنامه :

کل برنامه در یک کلاس Main پیاده‌سازی شده و شامل بخش‌های زیر است:

ایجاد پنجره اصلی (JFrame)

استفاده از CardLayout برای مدیریت چند صفحه

صفحه خوش‌آمدگویی (Intro)

صفحه محاسبه BMI (Calculator)

مدیریت رویدادها با ActionListener


جریان کلی اجرای برنامه:

اجرای برنامه و نمایش صفحه خوش‌آمدگویی

ورود به صفحه محاسبه با دکمه Start

وارد کردن وزن و قد

محاسبه BMI و نمایش نتیجه

دریافت اطلاعات تکمیلی و پیشنهادات سلامتی


*** شخصی‌سازی شده و امکانات بیشتر :

1_ محاسبه وزن ایده‌آل:

وزن ایده‌آل بر اساس BMI نرمال (عدد 22) و قد فعلی کاربر محاسبه می‌شود

به کاربر نشان می‌دهد چه وزنی برای قرار گرفتن در محدوده سالم مناسب است

2_ پیشنهادات ساده سلامتی:

بر اساس دسته‌بندی BMI، توصیه‌های کوتاه و کاربردی ارائه می‌شود

شامل پیشنهادات تغذیه‌ای و فعالیت بدنی

نمایش این اطلاعات در یک JOptionPane برای تعامل بهتر با کاربر
